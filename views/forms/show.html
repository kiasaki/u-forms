Usage

HTML Form Example

Sending data to μForms consists in creating an HTML form with the "action" attribute set to this form's unique URL. For example: 

<form method="post" action="https://{{app_base_url}}/f/{{form.id}}">
  <input type="email" name="email" placeholder="Your email">
  <textarea name="message" placeholder="Your message"></textarea>
  <button type="submit">Send</button>
  <input type="text" name="_bot" style="display:none">
</form>
Advanced features

By having an input with a "name" attribute set to _next you can provide an URL to redirect the user to after the form is submitted. By default μForms shows a "Thank You" page. 

<input type="hidden" name="_next" value="https://mysite.com/thank-you">

By having an input with a "name" attribute set to _email (or email) you can provide an email address to use as "Reply-To" value in the notification email sent to you. 

<input type="hidden" name="_email" value="some dynamic value">
- or -
<input type="email" name="email">
