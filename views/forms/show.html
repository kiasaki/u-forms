HTML Form Example

Sending data to Î¼Forms consists in creating an HTML form with the "action" attribute set to this form's unique URL. For example: 

<form method="post" action="https://{{app_base_url}}/f/{{form.id}}">
  <input type="email" name="email" placeholder="Your email">
  <textarea name="message" placeholder="Your message"></textarea>
  <button type="submit">Send</button>
</form>

By default, a "thank you" page is shown to the user after his form is submitted, in the event you want to user to go back to a specific page on your website, simply add a "hidden" input named "uforms_return_url" like below: 

<input type="hidden" name="uforms_return_url" value="https://mysite.com/thank-you">
